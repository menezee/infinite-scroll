# Infinite Scroll

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

* [Node.js](https://nodejs.org)

### Installing

```
$ git clone https://github.com/menezee/infinite-scroll.git

$ cd infinite-scroll

$ npm i

$ npm start

```
